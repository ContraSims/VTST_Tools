# VTST_Tools

VTST Tools (Transition State Tools for VASP) are a collection of scripts used to find saddle points and calculate reaction rates (Transition State Theory) within the VASP simulation package. Developed by the Henkelman group, they provide methods like NEB, Dimer, and Lanczos, along with optimizers and pre/post-processing scripts to model atomic-scale reactions and pathways.
